# CONTRIBUTING

## Принципы
- По умолчанию репозиторий приватный.
- Не коммитьте секреты, `.env`, токены, ключи, кэш, логи и локальные файлы прогресса.
- Не включайте `assessments/answer_keys` в web-сборку.
- Не удаляйте или не перезаписывайте качественно подготовленные материалы без явной причины.

## Стандарт модуля
Каждый модуль должен содержать:
- `README.md`
- `01-theory.md`
- `02-examples.md`
- `03-case-study.md`
- `04-practical-assignment.md`
- `05-quiz.md`
- `06-reflection.md`
- `resources.md`

Ключи ответов и рубрики хранятся отдельно в `assessments/`.

## Локальная проверка
```bash
npm run generate:manifest
npm run validate
```

## Коммиты
Используйте осмысленные сообщения:
- `chore: ...`
- `feat: ...`
- `fix: ...`
- `docs: ...`
- `ci: ...`
- `test: ...`
