# SECURITY

## Что защищаем
1. **Учебные материалы** — доступны в UI.
2. **Ключи ответов и внутренние рубрики** — не попадают в публичную web-сборку.
3. **Личный прогресс** — хранится локально в браузере через `localStorage`.

## Запрещено коммитить
- токены GitHub и API;
- `.env` и производные;
- SSH-ключи и сертификаты;
- локальные дампы прогресса;
- временные файлы, кэш и логи;
- конфигурации с секретами.

## Answer keys
Генератор манифеста сообщает только факт наличия ключа (`hasAnswerKey`). Содержимое ключей не включается ни в `course-manifest.json`, ни в `web/public/content`.

## Проверки
Перед публикацией выполняются проверки:
- структуры модулей;
- локальных Markdown-ссылок;
- отсутствия answer keys в публичном артефакте;
- корректности манифеста;
- отсутствия секретов в Git;
- отсутствия локальных пользовательских данных в Git.
